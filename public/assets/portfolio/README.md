Place portfolio images in this folder before running `npm run dev` or `npm run build`.

Suggested names:

- `styling.png` - portrait/photo of Gurdharam Jeet Singh
- `videogen.png` or `videogen.webp` - VideoGen preview/icon
- `doodhisaab.png` or `doodhisaab.webp` - DoodHisaab icon/preview
- `fasaldoctor.png` or `fasaldoctor.webp` - Fasal Doctor icon/preview

The app auto-detects images by filename keywords.
